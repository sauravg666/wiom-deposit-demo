Wiom Partner App : Security Deposit (v2)
Proposed new deposit model — no 50% reserve rule. Per-netbox security treated as discrete slots, with credits on return and clearer loss-handling.
What's Covered
  • Main deposit screen — active netboxes, return credits, forfeited count
  • Partner returns netboxes → credits added, deposit cash unchanged
  • Custodial loss confirmed by Wiom → full ₹1,538 deducted from wallet
  • Failed recovery → ₹200 security forfeited as a book entry (reversible)
  • Partner orders new netboxes → ₹200 each goes from wallet to deposit
